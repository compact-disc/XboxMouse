# Xbox Mouse
A simple, lightweight program that allows an Xbox One/360 controller to be used as a mouse.
<br>
<br>
Packages:
<br>
* SharpDX 4.2.0
* SharpDX.XInput 4.2.0
<br>

Controls:
<br>
* A is Left Click
* B is Right Click
* Y is Middle Click
* Left Thumbstick is Mouse Movement
* Right Thumbstick is Scrolling (Up/Down, Left/Right)
<br>
