# Xbox Mouse
A simple, lightweight program that allows an Xbox One/360 controller to be used as a mouse.
<br>
<br>
Packages:
<br>
* SharpDX 4.2.0
* SharpDX.XInput 4.2.0
<br>
<br>
Controls:
<br>
* A - Left Click
* B - Right Click
* Y - Middle Click
* Left Thumbstick - Mouse Movement
* Right Thumbstick - Scrolling (Up/Down, Left/Right)
